# Personal Portfolio — Deepika Gummalla

A modern, responsive, and interactive portfolio website showcasing my projects, skills, certifications, and academic journey as an AI/ML Engineer.

## Features
- **Interactive 3D ID Card**: A premium 3D tilt card effect with dynamic glare and auto-animations.
- **Particle Background**: A custom, lightweight interactive particle canvas system.
- **Scroll Reveal Animations**: Dynamic elements that fade and slide into view as you scroll.
- **Typing Animation**: A sleek typing effect showcasing core focus areas.
- **Responsive Design**: Clean and accessible layouts optimized for mobile, tablet, and desktop screens.
- **Project Showcase**: Displaying specialized projects in AI, LLM fine-tuning, and prompt engineering.

## Tech Stack
- **HTML5** (Semantic Structure & SEO Optimizations)
- **CSS3** (Custom Styling, CSS Grid/Flexbox, and Glassmorphism)
- **JavaScript** (Vanilla JS for interactive animations and custom logic)
